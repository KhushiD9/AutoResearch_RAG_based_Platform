                           🎯 WHAT IT DOES 🎯


1. USER SEARCHES → "machine learning"

2. BACKEND FINDS → Papers from Semantic Scholar & arXiv

3. DOWNLOADS & PROCESSES → PDFs automatically

4. EXTRACTS & CHUNKS → Text intelligently split

5. CREATES EMBEDDINGS → Converts to vectors (AI)

6. STORES IN DATABASE → ChromaDB vector storage

7. USER ASKS QUESTION → "What are the main findings?"

8. SEARCHES VECTORS → Finds most relevant sections

9. SENDS TO AI → Gemini/GPT with context

10. GETS ANSWER → Citation-backed response!

                        ⚡ QUICK START PREVIEW ⚡


Backend Setup (3 minutes):
    cd backend
    python -m venv venv
    venv\Scripts\activate                 (Windows)
    pip install -r requirements.txt
    copy .env.example .env
    # Add your Gemini API key to .env
    python main.py

Frontend Setup (2 minutes):
    cd frontend
    npm install
    npm start

Open Browser:
    http://localhost:4200

Done! Start searching for papers!
